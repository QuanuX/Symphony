# SACV Engine Intent

Provide an independently installable C++26 freezing-path engine that turns canonical SACV registry and owner-held OpenAPI truth into deterministic validation, compatibility evidence, caller-declared proposals, and disposable inventory projections.

The engine exists to complete SACV's programmatic circuit without moving endpoint semantics into the tool. It never creates an endpoint, decides ownership, writes canonical files, publishes documentation, generates runtime bindings, or opens a listener.
