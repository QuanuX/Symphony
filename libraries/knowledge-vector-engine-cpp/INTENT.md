# Knowledge Vector Engine C++ Foundation Intent

## Purpose

Provide authority-free, statically linked mechanics shared by independently installed Symphony knowledge-vector processes.

## Implemented Scope

The `0.1.0-dev` foundation implements bounded JSON parsing, strict process-envelope parsing, stable error codes, deterministic JSON response encoding and response digests, SHA-256, safe relative-path checks, no-follow regular-file reads, and deterministic bounded file snapshots.

## Non-Goals

The foundation does not decide vector membership, architectural meaning, feature-worthiness, compatibility, publication, permission, ratification, session policy, or canonical changes. It contains no network listener, dynamic plugin loader, provider client, canonical apply route, Maestro integration, SSIAG credential, or STAV writer.

## Authority

`knowledge/` owns the common protocol and lifecycle contracts. Each vector owns its domain semantics. This library is implementation truth subordinate to those contracts and cannot manufacture authority.
